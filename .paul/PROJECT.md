# Lipa Health

## Vision
Precision health platform for Europe. What your doctor won't tell you — 100+ biomarkers tested against optimal ranges, personalized research-backed protocols, and proof it works over time.

## Core Value
Democratize longevity medicine. Take what costs $50K/yr at a private clinic and deliver 80% of the value at EUR 30-50/mo through AI + lab partnerships.

## Target Customer
Health-conscious Europeans, 35-50, who want more than "normal" from their doctor. Biohackers, longevity enthusiasts, anyone running supplement stacks or peptide protocols who wants data to prove what's working.

## Revenue Model
- Testing subscription: EUR 200-300/year
- Protocol subscription: EUR 30-50/month
- Affiliate revenue: 10-15% on supplements/peptides purchased through recommendations
- Blended: ~EUR 75-95/month per user

## Markets (launch order)
1. Poland (home base, Diagnostyka partnership, zero competition, cheapest labs)
2. Netherlands (current residence, test market)
3. Germany (largest EU market, Aware is weak)
4. Spain (greenfield after Melio failed)
5. UK (last — most competitive)

## Competitive Positioning
- vs Lucis (YC, $8.5M): We do protocols + marketplace, they stop at data
- vs Function Health ($2.5B): We're in Europe, they're US-only
- vs Polish tools (ZDROT, BloodLab, Kantesti): We're a premium product, they're GPT wrappers
- vs Longevity+ (Poland): We're EUR 50/mo, they're EUR 6,000-22,000

## Key Differentiators
1. Research-backed protocols with confidence scores (nobody does this)
2. Peptide protocols connected to biomarkers (nobody does this)
3. Cross-marker pattern recognition (LLM-powered, not rule-based)
4. Affiliate marketplace for supplements + peptides
5. Longitudinal tracking with protocol adjustment
6. Wearable data integration

## Tech Stack
- Frontend: Next.js 14+ (App Router) + Tailwind CSS + shadcn/ui
- Database + Auth + Storage: Supabase
- AI: Claude API (interpretation + PDF vision parsing)
- Charts: Recharts or Nivo
- Payments: Stripe
- Wearables: Terra API
- Email: Resend
- Hosting: Vercel
- Domain: lipa.health
