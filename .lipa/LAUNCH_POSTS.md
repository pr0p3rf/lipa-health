## Hacker News — Show HN

**Title:** Show HN: Lipa – Upload your blood test, get research-grade analysis for €39

**Body:**
I built Lipa (https://lipa.health) because I was frustrated with how blood test results are communicated. Doctors have 5 minutes. ChatGPT hallucinates citations. InsideTracker charges $99+.

Lipa uploads your blood test PDF, extracts biomarkers via Claude, then cross-references each against a corpus of 100K+ PubMed studies using RAG (pgvector similarity search). Each marker gets:

- Root cause analysis connecting to your other markers
- Specific research with sample sizes, effect sizes, journal names
- Protocol-level recommendations (dose, form, timing, expected improvement)
- Cross-marker pattern detection
- Biological age estimate (KDM + PhenoAge ensemble)

Stack: Next.js, Supabase (pgvector), Claude Sonnet for analysis, OpenAI for embeddings.

Free preview shows all markers with status. Full analysis is €39 one-time.

Would love technical feedback and feature requests.
